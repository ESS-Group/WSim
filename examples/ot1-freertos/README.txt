
This directory contains freertos demo application.
Applications are derived from FreeRTOS Demos for MSP430f449
micro-controllers.

FreeRTOS source files are available from the FreeRTOS 
website http://www.freertos.org/

To install the demo you should download FreeRTOS3.2.4.zip and unzip the
archive to create the FreeRTOS3.2.4 directory.
